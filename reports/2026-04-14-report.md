## 🔬 重点论文详情

### 1. [PolyU × Meta] PhysInOne: Visual Physics Learning and Reasoning in One Suite
- **arXiv**: [2604.09415](http://arxiv.org/abs/2604.09415)
- **会议**: CVPR 2026
- **核心内容**: 超大规模物理数据集——200万视频、153,810个动态3D场景，覆盖71种物理现象（力学/光学/流体/磁学）
- **团队**: PolyU vLAR Group (Bo Yang 教授)，39人团队，与 Meta 和 Syai Singapore 合作
- **📌 关注点**: PolyU 在物理 AI / 世界模型方向的重大布局，规模超越所有现有数据集
- **PDF 验证**: ✅ PolyU 主导，Meta 合作

### 2. [HKUST + CityU] InsEdit: Instruction-based Visual Editing
- **arXiv**: [2604.08646](http://arxiv.org/abs/2604.08646)
- **核心内容**: 基于 HunyuanVideo-1.5 构建指令式视频编辑模型，仅用约10万视频编辑数据即达 SOTA
- **团队**: HKUST 陈启峰组 + Celia Research HK + CityU
- **关键技术**: Mutual Context Attention (MCA)，支持从视频中间开始编辑
- **📌 关注点**: 虽基于腾讯 HunyuanVideo，但由港校独立完成的开源研究
- **PDF 验证**: ✅ HKUST + CityU + Celia Research HK

### 3. [HKU + CUHK] AniGen: Unified S³ Fields for Animatable 3D Asset Generation
- **arXiv**: [2604.08746](http://arxiv.org/abs/2604.08746)
- **发表**: SIGGRAPH 2026 (ACM TOG)
- **核心内容**: 统一框架直接从单张图片生成可动画3D资产（几何+骨骼+蒙皮权重）
- **团队**: HKU 戚晓娟 CVMI Lab + VAST + CUHK + 清华
- **📌 关注点**: HKU 与 VAST 的深度合作，3D 生成前沿
- **PDF 验证**: ✅ HKU 主导 (4人)，VAST (3人)，CUHK (1人)

### 4. [HKU] SATO: Strips as Tokens — Artist Mesh Generation
- **arXiv**: [2604.09132](http://arxiv.org/abs/2604.09132)
- **发表**: SIGGRAPH 2026 Journal Track (ACM TOG)
- **核心内容**: 基于三角形条带的 token 排序策略统一三角形和四边形 mesh 生成
- **团队**: HKU CGVU Lab (Taku Komura) + ShanghaiTech + Deemos + 山大
- **第一作者**: Rui Xu (徐瑞) — **SIGGRAPH 2023 最佳论文奖得主**
- **PDF 验证**: ✅ HKU 主导
